# 🔥 Naruto AR – Real-Time Hand Gesture Web App

An interactive web-based Augmented Reality (AR) project that detects hand gestures in real-time and triggers dynamic Naruto-themed visual effects.

Built using computer vision with MediaPipe Hands and rendered using HTML, CSS, and JavaScript.

## 🧠 Features

- ✋ Real-time hand tracking using MediaPipe Hands
- 🔥 Open palm gesture detection
- ⚡ Smooth power charge animation

## 🎯 How It Works

1. Camera stream is captured using MediaPipe Camera Utils
2. MediaPipe Hands detects 21 hand landmarks
3. Gesture logic checks finger positions
4. When open palm is detected:
   - Power value increases
   - Visual effect is triggered
5. Effect fades when hand closes or disappears
